- [ ] 포트포워딩 승인 완료

- [ ] 외부 포트 ↔ 내부 포트 매핑 확정

## Plane (서버에 이미 구축 완료)

- [x] 저장소 클론
- [x] 환경 파일 생성 및 내부 IP(172.20.4.60) 설정

- [x] 프론트엔드 빌드 및 실행

## 이관 시 주의사항

- Windows 테스트 데이터는 이관하지 않음 (서버에서 새로 시작)
- .env 파일의 localhost → 172.20.4.60 변경 필수
- GitLab external_url도 서버 IP로 변경 필요
- 서버의 inotify 제한이 적용 안 되므로 `pnpm dev` 대신 `pnpm build && pnpm start` 사용
