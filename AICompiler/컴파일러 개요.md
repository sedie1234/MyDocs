# 컴파일러 개요

**최종 갱신**: 2026-03-30

---

## 프로젝트 구조

Infetron-V2 NPU를 위한 AI 컴파일러. 두 가지 배포 파이프라인이 존재.

```
                    ┌── Independent Compiler (independent/)
                    │     StableHLO 레벨 양자화 + 최적화
AI Model ──────────┤     현재 활성 개발 중
                    │
                    └── IREE Plugin (to_iree/ + repH)
                          IREE 기반 lowering (INF_CAP → INF_ISA → INF_EXP)
                          ISA 미확정으로 대기 중
```

### Independent Compiler

StableHLO IR 위에서 동작하는 독립 컴파일러.
IREE에 의존하지 않고, MLIR upstream + StableHLO pass만 사용.

- **입력**: QDQ ONNX → StableHLO MLIR
- **파이프라인**: [[QDQ Direct 파이프라인]] (Step0~Step5)
- **출력**: quant type annotated StableHLO (conv 83/83 i8)
- **도구**: `ind-opt` (개별 pass), `ind-compile` (파이프라인)
- **위치**: `pubH/independent/`

### IREE Plugin

IREE 컴파일러에 Infetron-V2 전용 plugin을 추가하는 방식.
repH(하드웨어 IR)를 submodule로 포함.

- **입력**: ONNX → IREE import → linalg
- **파이프라인**: [[INF_CAP Layer]] → [[Custom IR 설계|INF_ISA]] → INF_EXP
- **출력**: `out_bin.ihnn`, `plan.json`
- **도구**: `iree-opt`, `iree-compile`
- **위치**: `pubH/to_iree/`, `pubH/third_party/RepresentHW/`
- **상태**: ISA 미확정으로 대기 (2026-03-16~)

## 현재 개발 상태

| 컴포넌트 | 상태 | 브랜치 |
|---|---|---|
| Independent Step1~4 | **확정** | `main` |
| Independent Step5 | **확정** | `step5-quantize-region` |
| IREE INF_CAP Pass | 완료 (8개 Pass) | `main` |
| IREE INF_ISA | 대기 (ISA 미확정) | - |
| IREE INF_EXP | 대기 | - |
