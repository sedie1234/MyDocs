## 사용 팁

1. **간단 버전부터 시작**: 각 스킬에 "간단 버전" 프롬프트가 있음. 빠른 답이 필요하면 이것부터.
2. **계산 과정 요청**: "계산 과정을 보여줘"를 추가하면 어떤 값이 어떻게 사용되었는지 전개.
3. **복수 가정**: 여러 변경을 동시에 분석하고 싶으면 `skill_04_spec_impact.md`의 비교 분석 버전 사용.
4. **회의 중 빠른 계산**: `skill_07_quick_calc.md`의 예시를 참고하여 짧은 질문.
